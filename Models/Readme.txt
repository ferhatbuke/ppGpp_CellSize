Jupyter codes for simulating 1000 cells under a ppGpp shift.

Naming of the models represent the experiments for which they are configured.

Pos: Positive model (direct model as described in the publication)
Neg: Negative model (hierarchical model as described in the publication)

R: Rich media
M: Minimal media

1ng: 1ng/ml DOX induction
2ng: 2ng/ml DOX induction
